## 🔄 Migration Guide

### From JSON to JSONL
1. Set `"storage_backend": "jsonl"` in config
2. Set `"auto_migrate": true` in config
3. Restart PicoClaw
4. Sessions automatically migrated on first access
5. Old JSON files preserved as backup

### Enabling Model Routing
1. Set `"routing.enabled": true` in config
2. Configure model tiers
3. Restart PicoClaw
4. Monitor logs for routing decisions

### Using PICOCLAW_HOME
1. Set `PICOCLAW_HOME=/custom/path` environment variable
2. Restart PicoClaw
3. All data stored in custom path
4. Old data in ~/.picoclaw remains untouched
