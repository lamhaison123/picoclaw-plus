## 🔄 Migration Path

### From JSON to JSONL
1. Enable auto-migration: `"auto_migrate": true`
2. Set backend: `"storage_backend": "jsonl"`
3. Restart PicoClaw
4. Sessions automatically migrated on first access
5. Old JSON files preserved as backup
