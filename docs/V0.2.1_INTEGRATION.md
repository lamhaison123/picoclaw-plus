## Quick Start

### Enable All Features
```json
{
  "routing": {
    "enabled": true,
    "tiers": [
      {"name": "cheap", "models": ["gpt-4o-mini"]},
      {"name": "expensive", "models": ["gpt-5.2"]}
    ]
  },
  "session": {
    "storage_backend": "jsonl",
    "auto_migrate": true
  },
  "tools": {
    "file_tools_enabled": true,
    "web_tools_enabled": true
  }
}
```

### Using .env
```bash
PICOCLAW_ROUTING_ENABLED=true
PICOCLAW_SESSION_STORAGE_BACKEND=jsonl
OPENAI_API_KEY=sk-...
```
